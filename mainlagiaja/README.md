## Run Locally

Clone the project

```bash
  git clone https://github.com/farah2222/mainlagiaja
```

Go to the project directory

```bash
  cd my-mainlagiaja
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
